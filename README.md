# Musion — The Production Intelligence Layer

A landing page for Musion, the production intelligence platform for independent artists.

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Supabase (waitlist)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## About

Musion guides independent artists through every stage of music production — from Ideation to Distribution — with smart matching, structured workflows, and a data flywheel that gets smarter with every project.
